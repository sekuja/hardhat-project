# ROME TESTNET

`git clone https://github.com/sekuja/hardhat-project.git`

`cd hardhat-project`

`npm install`

Edit the `.env` file, input your private key (Remove the `0x` prefix).
`PRIVATE_KEY=your_private_key_here`

`npx hardhat compile`

`npx hardhat run scripts/deploy.js --network rome`

Done
